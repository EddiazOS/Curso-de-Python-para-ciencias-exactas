## Descripción

Este curso es una adaptación pedagógica al español del [CHE597 – Data Science in Chemical Engineering](https://secquoia.github.io/PU_CHE597_DSinChemE/) de la Universidad de Purdue, creado por el Dr. **David E. Bernal Neira** bajo licencia CC-BY-4.0.

Diseñado para estudiantes de biología, química, física y ciencias afines de la Universidad de Cartagena que quieran aprender Python y ciencia de datos desde cero, sin conocimientos previos de programación.

## Estructura del curso

| Módulo | Tema | Estado |
|--------|------|--------|
| 0 | Primeros Pasos (Colab, notebooks, GitHub) | ✅ Listo |
| 1 | Fundamentos de Python | 🚧 En progreso |
| 2 | Análisis Numérico (NumPy, Matplotlib) | ⏳ Pendiente |
| 3 | Manejo de Datos (Pandas) | ⏳ Pendiente |
| 4 | Modelado y Estadística | ⏳ Pendiente |
| 5 | Aprendizaje Automático | ⏳ Pendiente |
| 6 | Aprendizaje Profundo | ⏳ Pendiente |
| 7 | Temas Avanzados (Quantum ML) | ⏳ Pendiente |
